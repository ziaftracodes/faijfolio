# How to Edit Your Portfolio

All text, images, links, and content can be edited in a single file:

üìù **`contentstoedit.ts`**

## What You Can Edit

- **Personal Information**: Name, role, tagline, photos
- **Navigation**: Brand name, menu links
- **Hero Section**: Badge, title, description, buttons
- **Work/Projects**: Titles, descriptions, images, tags, links
- **About Section**: Badge, title, cards, bio
- **Skills**: Badge, title, categories and skills
- **Request Work**: Title, description, offerings, stats
- **Contact**: Badge, title, description, email, social links
- **Footer**: Copyright text, social links

## How to Edit

1. Open `src/config/contentstoedit.ts`
2. Find the section you want to edit
3. Update the text, images, or links
4. Save the file - changes will appear automatically!

## Tips

- For images, use direct URLs (https://...) or local paths (/images/...)
- Keep the same data structure when editing
- Each section is clearly labeled with comments
